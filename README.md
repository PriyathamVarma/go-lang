# go-lang

## Steps

**Install Go**

1. Go to [link](https://go.dev/dl/)
2. Create a newfile -> hello.go
3. ```
	// Packages
	package main

	// Imports
	import "fmt"

	// Functions
	func main() {
		fmt.Println("hello world")
	}

	```

4. go to terminal > go run hello.go 

## Packages

Packages group certain files into a group 

1. Executable
2. Reusable : Helper
